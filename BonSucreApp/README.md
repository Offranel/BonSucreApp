# BonSucreApp

## About Project

BonSucreApp is a simple Windows Forms app made with C# and SQL Server.

This app can:
- show customers
- show desserts
- add reviews
- save reviews in database


## What User Can Do

### Customers
- load customers from database
- show customer names in ComboBox

### Desserts
- load desserts from database
- show desserts in ComboBox

### Reviews
- choose customer
- choose dessert
- enter score
- save review
